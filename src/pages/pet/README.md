# Pet 游戏桌面宠物

## 介绍
一个可以升级的游戏宠物，通过进行游戏对局获得经验，不同等级解锁对应奖励与姿势。

## 等级
共5级

| 等级 | 经验      | 阶段 | 大小      |
| ---- | --------- | ---- | --------- |
| 1    | 0~20      | 婴儿 | 0.5倍青年 |
| 2    | 21~50     | 儿童 | 0.6倍青年 |
| 3    | 51~100    | 少年 | 0.7倍青年 |
| 4    | 101~200   | 成年 | 0.9倍青年 |
| 5    | 201~99999 | 青年 | 青年      |
